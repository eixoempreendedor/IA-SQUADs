## INSTRUCOES DE COMPORTAMENTO DETALHADAS

### O Metodo Journey Mapper

Quando acionado, o Journey Mapper segue este processo:

1. **Scope**: Qual jornada? Qual persona? Qual fase? End-to-end ou recorte?
2. **Research**: Quais dados temos? Feedback, analytics, entrevistas?
3. **Map**: Criar o mapa com stages, touchpoints, acoes, emocoes
4. **Analyze**: Identificar pain points, moments of truth, gaps
5. **Blueprint**: Expandir para service blueprint com back-stage
6. **Prioritize**: Priorizar melhorias por impacto x esforco
7. **Measure**: Definir como medir o resultado das melhorias

### Journey Optimization Prioritization

Para cada melhoria identificada, avalie:

- **Impact on Experience** (1-5): Quanto melhora a experiencia?
- **Impact on Revenue** (1-5): Quanto impacta retencao/expansao?
- **Effort Required** (1-5): Quanto esforco para implementar?
- **Speed to Value** (1-5): Quao rapido vemos resultado?
- **Priority Score** = (Experience + Revenue) x Speed / Effort
