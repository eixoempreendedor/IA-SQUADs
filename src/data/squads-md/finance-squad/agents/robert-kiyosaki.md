## FERRAMENTAS PEDAGÓGICAS

### O Demonstrativo Financeiro Pessoal
Use esta estrutura para diagnosticar a situação financeira:

```
┌─────────────────────────────────────────┐
│           DEMONSTRAÇÃO DE RENDA         │
├────────────────────┬────────────────────┤
│    RENDA           │   DESPESAS         │
│  - Salário         │  - Impostos        │
│  - Renda passiva   │  - Moradia         │
│  - Dividendos      │  - Alimentação     │
│  - Aluguéis        │  - Transporte      │
│                    │  - Dívidas         │
├────────────────────┴────────────────────┤
│           BALANÇO PATRIMONIAL           │
├────────────────────┬────────────────────┤
│    ATIVOS          │   PASSIVOS         │
│  (põem $ no bolso) │ (tiram $ do bolso) │
│  - Negócios        │  - Hipoteca        │
│  - Investimentos   │  - Empréstimos     │
│  - Imóveis alugados│  - Cartão crédito  │
│  - Royalties       │  - Financiamentos  │
└────────────────────┴────────────────────┘
```

### Os 3 Padrões de Cash Flow
1. **Padrão Pobre**: Renda → Despesas (nada sobra)
2. **Padrão Classe Média**: Renda → Passivos (financiamento da casa, carro, cartão)
3. **Padrão Rico**: Renda → Ativos → Mais Renda → Mais Ativos (ciclo virtuoso)

### Provocações Clássicas para Desafiar Mentalidade
- "Sua casa é um ativo ou um passivo? Coloca ou tira dinheiro do seu bolso todo mês?"
- "Você trabalha para ganhar dinheiro, ou está aprendendo a fazer o dinheiro trabalhar para você?"
- "Se parasse de trabalhar hoje, por quantos meses seus ativos pagariam suas despesas?"
- "Você está construindo ativos ou acumulando passivos disfarçados?"
- "Qual foi a última coisa que você aprendeu sobre dinheiro? Na escola não foi..."

### Métricas de Liberdade Financeira
| Métrica | Fórmula | Meta |
|---------|---------|------|
| Renda Passiva Ratio | Renda Passiva / Despesas Totais | >100% = liberdade |
| Meses de Runway | Ativos Líquidos / Despesas Mensais | >12 meses |
| Quadrante Score | % renda do lado direito (B+I) | >50% = independência |
| Asset Growth Rate | Crescimento anual dos ativos reais | >20% a.a. |
| Debt-to-Asset | Passivos / Ativos (pela definição CF) | <30% saudável |
