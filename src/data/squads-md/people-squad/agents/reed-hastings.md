## GUIA DE IMPLEMENTACAO

### Avaliacao de Prontidao

Antes de implementar o modelo Netflix, avalie:

| Dimensao | Pronto | Nao Pronto |
|----------|--------|------------|
| Talent Density | >70% A-players | <50% A-players |
| Confianca | Alta confianca entre times | Cultura de blame |
| Lideranca | Lideres confortaveis com ambiguidade | Lideres controladores |
| Mercado | Industria de inovacao/criatividade | Industria altamente regulada |
| Tamanho | Qualquer (com adaptacoes) | N/A |

### Sequencia de Implementacao

**Fase 1: Talent Density (Meses 1-3)**
- Calibre o time atual com o Keeper Test
- Ajuste compensacao para top of market
- Inicie contratacoes de A-players

**Fase 2: Candor (Meses 3-6)**
- Implemente feedback 4A
- Inicie Sunshine Meetings
- Lideres modelam vulnerabilidade

**Fase 3: Remove Controls (Meses 6-12)**
- Simplifique politicas de despesas
- Implemente ferias flexiveis
- Substitua aprovacoes por principios

### O Keeper Test na Pratica

**Perguntas auxiliares para o Keeper Test:**
1. "Se esta pessoa saisse hoje, qual seria meu nivel de preocupacao? (1-10)"
2. "Eu contrataria esta pessoa de novo sabendo o que sei hoje?"
3. "Esta pessoa eleva o padrao do time ou e carregada pelo time?"
4. "O que aconteceria com os resultados do time sem esta pessoa?"

**Apos o Keeper Test:**
- Score 8-10: Invista pesadamente, de mais responsabilidade
- Score 5-7: Feedback claro + plano de 90 dias + re-avaliar
- Score 1-4: Generous severance + transicao digna

### Anti-patterns para Evitar

1. **Netflix Cosplay**: Copiar as praticas sem ter a talent density — resultado: caos
2. **Freedom Without Context**: Dar liberdade sem alinhar estrategia — resultado: desalinhamento
3. **Keeper Test Weaponized**: Usar como ameaca em vez de ferramenta de clareza — resultado: medo
4. **Premature Optimization**: Remover controles antes de ter a cultura — resultado: abuso
5. **One Size Fits All**: Aplicar igualmente para todos os niveis e funcoes — resultado: frustração
